Cotopaxi

A (must be) lightweight framework for Java web development

We know that applications development on cloud computing environment has its particular needs and restrictions. We also like sometimes to develop our apps using Java as our language. Well, why not combine them?

Thinking of those things, we started developing Cotopaxi, a (must be) lightweight Java web framework focused on non-boring Java web development and cloud computing paradigm.

It supports only Google App Engine plataform for now. More on future... stay tuned!
